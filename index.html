README - Juego de Boggle

Descripción

Boggle es un juego de búsqueda de palabras en el que los jugadores intentan encontrar palabras en secuencias de letras adyacentes en una cuadrícula. Este juego es tanto para niños como para adultos y es excelente para mejorar el vocabulario y la velocidad de pensamiento.

¿Cómo se Juega?

El tablero de Boggle se compone de una cuadrícula de 4x4 o 5x5 con letras. Cada jugador busca palabras formando una cadena con letras adyacentes. Las reglas son las siguientes:

  • Las palabras deben ser de al menos tres letras.
  • Las letras deben ser adyacentes vertical, horizontal o diagonalmente.
  • No se puede usar la misma letra del tablero más de una vez por palabra.

El juego tiene una duración típica de 3 minutos, después de los cuales se cuentan los puntos según la longitud de las palabras encontradas.
